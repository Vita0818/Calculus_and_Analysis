### 目录同步机制
- 前端页面只读取仓库内静态缓存文件：`data/drive-index.json`。
- Google Drive 目录树由 GitHub Actions 定时任务后台同步更新，而不是用户访问时实时扫描。
- 因此上传到 Google Drive 后，网站通常不会立刻显示，需等待下一次同步。
